# Roadmap

## Milestones

### M1 – Bootstrap e Auth ✅
- Estrutura modular por domínio
- Autenticação Supabase (login/registro)
- Bootstrap de usuário por role (business/creator)
- Rotas protegidas com AuthGuard

### M2 – Dashboards e Perfis
- Dashboard de empresa (stats, jobs, gráficos, recomendações)
- Dashboard de criadora (stats, ofertas)
- Perfil de empresa (edição, portfólio, mídia)
- Perfil de criadora (visualização)

### M3 – Descoberta e Jobs
- Mapa de criadoras (Google Maps)
- Criação de job com preço fixo + transporte
- Chat e pagamento (MVP mock)

### M4 – Refinamentos
- Testes E2E expandidos
- Otimizações de performance
- Ajustes de UX baseados em feedback
